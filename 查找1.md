## 考虑的基本数据结构

第一类： 查找有无--set
元素'a'是否存在，通常用set：集合
set只存储键，而不需要对应其相应的值。
set中的键不允许重复

第二类： 查找对应关系(键值对应)--dict
元素'a'出现了几次：dict-->字典
dict中的键不允许重复

第三类： 改变映射关系--map通过将原有序列的关系映射统一表示为其他。
## 典型例题
【两个数组的交集】（力扣349）
给定两个数组，编写一个函数来计算它们的交集。
示例 1：
输入：nums1 = [1,2,2,1], nums2 = [2,2]
输出：[2]
示例 2：
输入：nums1 = [4,9,5], nums2 = [9,4,9,8,4]
输出：[9,4]
说明：
输出结果中的每个元素一定是唯一的。
我们可以不考虑输出结果的顺序。
```python
class Solution:
    def intersection(self, nums1: List[int], nums2: List[int]) -> List[int]:
        dic = {}
        for num11 in nums1:
            if num11 not in dic:
                dic[num11] = 1
        
        res = []
        for num12 in nums2:
            if num12 in dic:
                res.append(num12)
        
        res = set(res)
        return res

class Solution:
    def intersection(self, nums1: List[int], nums2: List[int]) -> List[int]:
        set1 = set (nums1)
        set2 = set (nums2)
        return set1 & set2
```
